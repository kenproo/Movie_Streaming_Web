# ChillFilm - Movie Streaming Web App

## Tech Stack

- Frontend: React.js
- Backend: Spring Boot
- Database: MySQL
- Storage: AWS S3 / demo images
- CI/CD: GitHub Actions
- Deployment: Vercel + AWS / Docker

## Project Structure

- frontend/: React frontend
- backend/: Spring Boot backend
- docs/: project documentation

## Run Frontend

cd frontend
npm install
npm run dev

## Run Backend

cd backend
mvn spring-boot:run
